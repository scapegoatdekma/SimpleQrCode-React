
# QR Code Generator
This is an example React + Vite QR Code Generator. 

## Run Locally  

Clone the project  

~~~bash  
   git clone https://github.com/scapegoatdekma/SimpleQrCode-React
~~~

Go to the project directory  

~~~bash  
  cd QrCode-React
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run start
~~~
